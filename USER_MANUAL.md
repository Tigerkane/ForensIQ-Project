# ForensIQ User Manual

## Overview
ForensIQ is an offline investigation intelligence platform powered by locally running Large Language Models (LLMs). It extracts structured intelligence, evidence, timelines, and relationships from raw investigative documents.

## Installation
1. Install Python 3.12+ and Ollama.
2. Run the backend API via `uvicorn main:app`.
3. Access the web interface using the Live URL or run the frontend locally using `npm run dev`.

## Usage
1. Drag and drop evidence files into the upload box.
2. Select the intelligence model.
3. Click "Extract Intelligence".
4. Review the timeline, relationships, and extracted insights.
